# MiniHelmets 🏁

**Versión:** 0.0.0  

Proyecto **MiniHelmets**, una tienda online de mini cascos coleccionables de F1 y MotoGP.  
Este repositorio contiene dos partes principales:

- `frontend/`: Interfaz de usuario.
- `backend/`: API y lógica del negocio.
